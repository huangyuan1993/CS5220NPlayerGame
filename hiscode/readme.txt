Attached is everything for n-player games.  You will need to download
or compile phc as well as configure and mape google hash maps for your
architecture.  Going in to the sparsehash folder and running
"./configure && make" will probably do the trick.

http://www.math.uic.edu/~jan/download.html -- has executable versions
for many platforms.  The phcPack executable must be in a PATH
directory or the same folder as the nplayer_se exectauble.
